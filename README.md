# ETH Todo-List

Todo-list on eth blockchain

## Pre requisites: 

**Truffle Ganache** (local blockchain)
**Truffle** v5.0.2 (core: 5.0.2)
**Solidity** v0.5.0 (solc-js)
**Node** v18.8.0

## Run

* `truffle migrate --reset`
* `npm run dev`

Additionaly I had to install truffle contract
`npm i @truffle/contract`
